# 📚 LEET 지식 맵 — 배포 가이드

코딩 경험이 없어도 할 수 있어요! 순서대로 따라해주세요.
총 소요시간: 약 20~30분

---

## ✅ STEP 1 — GitHub 가입 (5분)

1. https://github.com 접속
2. 우측 상단 "Sign up" 클릭
3. 이메일, 비밀번호, 사용자명 입력해서 가입
4. 이메일 인증 완료

---

## ✅ STEP 2 — GitHub에 프로젝트 올리기 (5분)

1. 로그인 후 우측 상단 "+" 버튼 → "New repository" 클릭
2. Repository name: `leet-study-app` 입력
3. "Public" 선택 (Vercel 무료 사용을 위해)
4. "Create repository" 클릭
5. 새 페이지에서 "uploading an existing file" 링크 클릭
6. 이 zip 파일을 압축 해제한 폴더 전체를 드래그앤드롭
7. 하단 "Commit changes" 클릭

---

## ✅ STEP 3 — Vercel로 배포 (5분)

1. https://vercel.com 접속
2. "Sign Up" → "Continue with GitHub" 클릭 (GitHub 계정으로 로그인)
3. "New Project" 클릭
4. 방금 만든 "leet-study-app" 선택 → "Import" 클릭
5. 설정은 건드리지 말고 "Deploy" 클릭
6. 1~2분 기다리면 배포 완료!
7. `https://leet-study-app-xxx.vercel.app` 같은 링크 생성됨

---

## ✅ STEP 4 — API 키 발급 및 설정 (10분)

### API 키 발급
1. https://console.anthropic.com 접속
2. 구글 계정 등으로 가입/로그인
3. 좌측 메뉴 "API Keys" 클릭
4. "Create Key" 클릭 → 이름 입력 (예: leet-app)
5. 생성된 키 복사 (sk-ant-... 로 시작하는 긴 문자열)
   ⚠️ 이 키는 다시 볼 수 없으니 메모장에 임시 저장!

### Vercel에 API 키 입력
1. Vercel 대시보드에서 "leet-study-app" 프로젝트 클릭
2. 상단 "Settings" 탭 클릭
3. 좌측 메뉴 "Environment Variables" 클릭
4. Name: `REACT_APP_ANTHROPIC_API_KEY`
5. Value: 복사한 API 키 붙여넣기
6. "Save" 클릭
7. 상단 "Deployments" 탭 → 최신 항목 우측 "..." → "Redeploy" 클릭

---

## 🎉 완료!

이제 링크를 사무실 컴퓨터, 집 컴퓨터, 스마트폰 어디서든
브라우저로 접속하면 LEET 지식 맵을 사용할 수 있습니다!

링크는 Vercel 대시보드에서 항상 확인 가능합니다.

---

## ❓ 문제가 생겼을 때

- AI 문제가 생성 안 됨 → API 키가 올바르게 입력됐는지 확인
- 페이지가 안 열림 → Vercel 배포 상태 확인 (빨간색이면 오류)
- 기타 문제 → Claude에게 오류 메시지를 복사해서 물어보세요!
